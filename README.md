# 交互式图像滤波应用

## 项目简介

这是一个基于Python和Streamlit的交互式图像滤波应用，实现了空间域滤波、图像梯度和频率域滤波三大核心功能。

## 功能特性

### 1. 空间域滤波
- 方框滤波（均值滤波）：可调节核大小
- 高斯滤波：可调节核大小和σ值
- 中值滤波：可调节核大小
- Sobel边缘检测：支持X方向、Y方向和融合梯度

### 2. 图像梯度
- 支持区域选择
- 计算梯度幅值和方向
- 可视化梯度结果
- 显示梯度统计信息

### 3. 频率域滤波
- 傅里叶变换：显示频谱幅值和相位谱
- 多种频域滤波器：
  - 理想低通/高通
  - 高斯低通/高通
  - 巴特沃斯低通/高通
- 支持参数调节

## 技术栈

- Python 3.8+
- OpenCV：图像处理
- NumPy：数值计算
- Matplotlib：数据可视化
- Streamlit：Web应用开发
- SciPy：科学计算
- Pillow：图像处理

## 安装与运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
streamlit run app.py
```

### 3. 访问应用

运行后，在浏览器中访问显示的URL（通常是 http://localhost:8501）

## 使用说明

1. **上传图像**：点击左侧上传区域，选择本地图像文件
2. **选择功能**：在顶部标签页中选择要使用的功能
3. **调节参数**：根据需要调节滤波器参数
4. **查看结果**：实时查看处理结果
5. **保存结果**：点击保存按钮保存处理结果

## 注意事项

- 支持的图像格式：jpg、jpeg、png、bmp
- 建议使用分辨率适中的图像，过大的图像可能影响处理速度
- 频域滤波处理时间较长，请耐心等待
